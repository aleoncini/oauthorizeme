OAuthorizeMe

A very simple web app to work with OAuth2 framework





To test the OAuth framework, specifically the Authorization Code Grant with PKCE, we will try to access the list of the repos of your GITHUB account.

To do it this application will redirect you to the authorization page of github and will require an AUTHORIZATION CODE for a specific client (that I have named oauthorizeme).

In order to obtain the code insert the CLIENT ID and press login button.

AUTHORIZATION CODE:

Very well, You have received an authorization code, probably after a succesful log in. This code can now be exchanged to obtain an ACCESS Token to operate with GITHUB APIs and display a list of your repos.

The operation requires a client secret to be specified. Copy and paste the secret and press the "Access Token" button
