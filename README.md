# OAuthorizeMe
A very simple web app to work with OAuth2 framework

### how to use it

Java and Maven are required to build and run the application

1. go to the directory where you want clone this project:
   > git clone https://github.com/aleoncini/oauthorizeme.git
   > cd oauthorizeme
2. build the app using Maven:
   > mvn clean package
3. run the app locally:
   > java -jar target/quarkus-app/quarkus-run.jar
4. access the web page containing the application using your browser:
   > open http://localhost:8080/
